            Notion integration

            To sync tasks and Jira stories to Notion, you need an integration token.

            1. Go to notion.so/my-integrations
            2. Click + New integration
            3. Name it "Dashboard", select your workspace, click Save
            4. Copy the Internal Integration Token (starts with ntn_ or secret_)
            5. Paste it below

            You'll also need to share each database with the integration:
            Open the database in Notion → ··· menu → Connections → add your integration.
